## Section Guidelines

- **TL;DR**: Must be scannable in 10 seconds. No fluff.
- **Comparison**: Minimum 3 alternatives. Include learning curve and community columns.
- **Use Cases**: 3 examples minimum, each with "When" context + runnable code.
- **Cheat Sheet**: Table must be sortable/scannable. Patterns are 1-liners.
- **Bonus**: 1-2 niche use cases that surprise the reader.
